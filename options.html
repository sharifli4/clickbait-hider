CORS Setup (required)

Ollama must allow requests from browser extensions. Configure CORS by setting:

OLLAMA_ORIGINS=chrome-extension://*

Linux (systemd):

sudo mkdir -p /etc/systemd/system/ollama.service.d
echo '[Service]
Environment="OLLAMA_ORIGINS=chrome-extension://*"' | sudo tee /etc/systemd/system/ollama.service.d/override.conf
sudo systemctl daemon-reload
sudo systemctl restart ollama

macOS / Manual:

OLLAMA_ORIGINS="chrome-extension://*" ollama serve

Windows:

Set environment variable OLLAMA_ORIGINS to chrome-extension://* in System Properties, then restart Ollama.
